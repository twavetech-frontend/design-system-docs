# Avatars

아바타는 사용자나 조직을 시각적으로 식별하는 작은 이미지/이니셜입니다. 프로필, 댓글, 멤버 목록 등에서 사용합니다.

Figma 컴포넌트: `Avatar`, `Avatar group`, `Avatar label group`

---

## Sizes

xs(24px), sm(32px), md(40px), lg(48px), xl(56px), 2xl(64px) 여섯 가지 사이즈를 지원합니다.

  <div style={{ display: 'flex', alignItems: 'center', gap: 16 }}>
    <Avatar size="xs" src="https://i.pravatar.cc/120?img=1" name="김민준" />
    <Avatar size="sm" src="https://i.pravatar.cc/120?img=1" name="김민준" />
    <Avatar size="md" src="https://i.pravatar.cc/120?img=1" name="김민준" />
    <Avatar size="lg" src="https://i.pravatar.cc/120?img=1" name="김민준" />

  </div>
</ComponentPreview>

## Avatar Group

여러 아바타를 겹쳐서 표시합니다. `max`로 표시할 개수를 제한하면 나머지는 `+N`으로 카운트됩니다.

<ComponentPreview code={`<AvatarGroup size="md" max={5} avatars={[...]} />`}>
  <div style={{ display: 'flex', flexDirection: 'column', gap: 16, alignItems: 'flex-start' }}>
    <AvatarGroup size="xs" max={5} avatars={sampleAvatars} />
    <AvatarGroup size="sm" max={5} avatars={sampleAvatars} />
    <AvatarGroup size="md" max={5} avatars={sampleAvatars} />
    <AvatarGroup size="lg" max={5} avatars={sampleAvatars} />
  </div>
</ComponentPreview>

### With add button

<ComponentPreview code={`<AvatarGroup size="md" max={4} showAddButton avatars={[...]} />`}>
  <div style={{ display: 'flex', flexDirection: 'column', gap: 16, alignItems: 'flex-start' }}>

## Figma Variants

| Property | Values |
| --- | --- |
| **Size** | `xs` `sm` `md` `lg` `xl` `2xl` |
| **Placeholder** | `True` `False` |
| **Text** | `True` `False` (이니셜 표시 여부) |
| **Status icon** | `False` `Online indicator` `Company` `Verified` |
| **Breakpoint** | `Desktop` `Mobile` |
