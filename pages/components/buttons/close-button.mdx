# Close Button

모달, 토스트, 배너 등을 닫는 데 사용하는 X 버튼입니다.
라이트/다크 배경 모드를 지원합니다.

Figma 컴포넌트: `Buttons/Button close X`

---

## Size

3가지 사이즈를 지원합니다: `sm` `md` `lg`.

<ComponentPreview code={`<Button variant="close" size="sm" iconOnly={true}>✕</Button>
