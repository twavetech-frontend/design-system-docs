# Dropdowns

드롭다운은 서브뷰에서 액션을 그룹화할 때 사용합니다.
메뉴를 숨기거나 여러 액션을 하나로 묶어야 할 때 유용합니다.

Figma 컴포넌트: `Base components/Dropdowns`

---

## Dropdown Trigger

드롭다운을 열기 위한 트리거 버튼입니다.

<ComponentPreview code={`<Dropdown trigger="button" label="Account" />
<Dropdown trigger="icon" />
<Dropdown trigger="avatar" avatarSrc="https://i.pravatar.cc/80?img=5" />`}>

  <Dropdown trigger="icon" />

</ComponentPreview>

## Figma Variants

### Dropdown

| Property | Values |
| --- | --- |
| **Type** | `Button` `Icon` `Avatar` |
| **Open** | `True` `False` |

### _Dropdown list item

Dropdown 메뉴의 개별 항목. 체크박스 변형과 구분선(Divider) 변형을 포함합니다.

| Property | Values |
| --- | --- |
| **Icon** | `True` `False` |
| **Checkbox** | `True` `False` |
| **State** | `Default` `Hover/active` `Disabled` |
| **Divider** | `True` `False` |

### _Dropdown list header

Dropdown 메뉴 상단에 배치되는 헤더. 사용자 정보(Avatar group) 또는 단순 헤더(Header) 두 가지 타입을 지원합니다.

| Property | Values |
| --- | --- |
| **Type** | `Avatar group` `Header` |
