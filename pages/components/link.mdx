# Link

텍스트 형태의 링크 컴포넌트입니다. 본문 흐름 중간, 푸터, 카드 등 별도의 컨테이너 없이 액션을 제공할 때 사용합니다.
외부 URL로 이동하는 경우 오른쪽에 `external-link` 아이콘을 함께 표시합니다.

Figma 컴포넌트: `Base components → Link`

---

## Color

브랜드 컬러 / 그레이 / Destructive 3가지 컬러를 제공합니다.

<ComponentPreview code={`<Link color="brand">Click to Download</Link>
<Link color="gray">Click to Download</Link>
<Link destructive>Click to Download</Link>`}>
  <Link color="brand">Click to Download</Link>
  <Link color="gray">Click to Download</Link>
  <Link destructive>Click to Download</Link>
</ComponentPreview>

| Color | 용도 |
| --- | --- |
| **brand** (`color="brand"`) | 기본 링크. 브랜드 보라(`#5200b0`). 본문 중간 강조 |
| **gray** (`color="gray"`) | 톤을 죽인 보조 링크. 카드·테이블 내부 등 시각적 무게가 낮아야 할 때 |
| **destructive** (`destructive`) | 삭제·경고 관련 링크. 항상 **밑줄** 표시(`#d92d20`) |

  <Link color="brand" size="lg">Click to Download</Link>
  <Link color="brand" size="xl">Click to Download</Link>

| Size | Font | Line height | Token |
| --- | --- | --- | --- |
| **sm** | 12px | 18px | `text-xs` |
| **md** (기본값) | 14px | 20px | `text-sm` |
| **lg** | 16px | 24px | `text-md` |
| **xl** | 18px | 28px | `text-lg` |

## Icon 없는 경우

`showIcon={false}`로 설정하면 external-link 아이콘을 숨길 수 있습니다. 내부 페이지 네비게이션 링크에 적합합니다.

<ComponentPreview code={`<Link color="brand" showIcon={false}>Read more</Link>
<Link color="gray" showIcon={false}>Read more</Link>
<Link destructive showIcon={false}>Delete account</Link>`}>
  <Link color="brand" showIcon={false}>Read more</Link>
  <Link color="gray" showIcon={false}>Read more</Link>
  <Link destructive showIcon={false}>Delete account</Link>
</ComponentPreview>
