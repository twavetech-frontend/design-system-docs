# Progress Indicators

진행 상태를 시각적으로 표현하는 컴포넌트입니다. 가로 막대 형태의 `Progress bar`와 원형 형태의 `Progress circle`을 제공합니다.

Figma 컴포넌트: `Progress bar`, `Progress circle`

---

## Progress Bar

### Label variants

라벨 위치를 5가지 형태로 설정할 수 있습니다.

#### No label

라벨 없이 진행바만 표시합니다.

<ComponentPreview code={`<ProgressBar value={0} label={false} />
<ProgressBar value={40} label={false} />
<ProgressBar value={80} label={false} />`}>
  <div style={{ display: 'flex', flexDirection: 'column', gap: 16, width: 320 }}>
    <ProgressBar value={0} label={false} />
    <ProgressBar value={40} label={false} />
    <ProgressBar value={80} label={false} />
  </div>
</ComponentPreview>

#### Right

진행바 우측에 % 텍스트를 표시합니다.

<ComponentPreview code={`<ProgressBar value={0} label="right" />
<ProgressBar value={40} label="right" />
<ProgressBar value={80} label="right" />`}>
  <div style={{ display: 'flex', flexDirection: 'column', gap: 16, width: 320 }}>
    <ProgressBar value={0} label="right" />
    <ProgressBar value={40} label="right" />
    <ProgressBar value={80} label="right" />
  </div>
</ComponentPreview>

#### Bottom

진행바 하단에 % 텍스트를 표시합니다.

<ComponentPreview code={`<ProgressBar value={0} label="bottom" />
<ProgressBar value={40} label="bottom" />
<ProgressBar value={80} label="bottom" />`}>
  <div style={{ display: 'flex', flexDirection: 'column', gap: 24, width: 320 }}>
    <ProgressBar value={0} label="bottom" />

    <ProgressBar value={80} label="bottom" />
  </div>
</ComponentPreview>

#### Top floating

진행률 위치 위에 작은 라운드 박스로 표시합니다.

<ProgressCircle size="xs" value={40} />
<ProgressCircle size="sm" value={40} />
<ProgressCircle size="md" value={40} />

| --- | --- |

| **Label** | `False` `Right` `Bottom` `Top floating` `Bottom floating` |

### Progress circle

| Property | Values |
| --- | --- |
| **Shape** | `Circle` `Half circle` |
| **Size** | `xxs` `xs` `sm` `md` `lg` |
