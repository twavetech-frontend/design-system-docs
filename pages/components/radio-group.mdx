# Radio Group

라디오 그룹은 여러 선택지 중 하나만 고를 수 있는 입력 컴포넌트입니다. 각 옵션은 카드 형태로 제목, 설명, 가격 등 부가 정보를 표시할 수 있습니다.

Figma 컴포넌트: `Radio group`, `Radio group item`

---

## Radio button

가장 기본 형태로, 라디오 버튼 옆에 제목·서브타이틀·설명을 함께 보여줍니다.

<ComponentPreview code={`<RadioGroup type="radio" items={[
  { value: 'basic', title: '기본 플랜', subTitle: '월 $10', description: '최대 10명...', selected: true },
  { value: 'business', title: '비즈니스 플랜', subTitle: '월 $20', description: '최대 20명...' },
]} />`}>
  <RadioGroup type="radio" size="sm" items={planItems} />
</ComponentPreview>

### Size md

<ComponentPreview code={`<RadioGroup type="radio" size="md" items={planItems} />`}>
  <RadioGroup type="radio" size="md" items={planItems} />
</ComponentPreview>

---

## Icon simple

좌측에 아이콘 박스가 추가된 형태입니다. 카테고리나 옵션 종류를 시각적으로 강조할 때 사용합니다.

<ComponentPreview code={`<RadioGroup type="icon-simple" items={[
  { value: 'inbox', title: '받은편지함', description: '...', selected: true },
  { value: 'archive', title: '보관함', description: '...' },
]} />`}>
  <RadioGroup type="icon-simple" size="sm" items={simpleItems} />
</ComponentPreview>

---

## Icon card

상단 헤더에 큰 featured icon이 있고, 하단 본문에 가격/설명/뱃지를 보여주는 확장 카드 형태입니다.

<ComponentPreview code={`<RadioGroup type="icon-card" items={[
  {
    value: 'basic',
    title: '기본 플랜',
    price: '$10',
    pricePeriod: '/월',
    description: '최대 10명...',
    badge: '기간 한정 할인',
    selected: true
  },
  ...
]} />`}>
  <RadioGroup type="icon-card" size="sm" items={cardItems} />
</ComponentPreview>

---

## Disabled

`disabled: true`로 항목을 비활성화 할 수 있습니다.

<ComponentPreview code={`<RadioGroup type="radio" items={[
  { value: 'a', title: '사용 가능한 옵션', description: '선택할 수 있습니다.', selected: true },
  { value: 'b', title: '비활성화된 옵션', description: '선택할 수 없습니다.', disabled: true },
]} />`}>
  <RadioGroup type="radio" size="sm" items={[
    { value: 'a', title: '사용 가능한 옵션', subTitle: '월 $10', description: '선택할 수 있습니다.', selected: true },

  ]} />
</ComponentPreview>
