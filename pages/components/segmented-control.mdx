# Segmented Control

세그먼티드 컨트롤은 서로 배타적인 옵션 중 하나를 선택하는 컴포넌트입니다. 짧은 옵션 그룹에서 현재 선택을 명확히 보여줄 때 적합합니다.

Figma 컴포넌트: `Segmented_control`

---

## Hug

콘텐츠 너비에 맞춰 좁게 표시되는 기본 형태입니다.

<ComponentPreview code={`<SegmentedControl size="md" items={[
  { label: '전체', current: true },
  { label: '진행중' },
  { label: '완료' },
]} />`}>
  <div style={{ display: 'flex', flexDirection: 'column', gap: 16, alignItems: 'flex-start' }}>
    <SegmentedControl size="sm" items={threeItems} />
    <SegmentedControl size="md" items={threeItems} />
  </div>
</ComponentPreview>

---

## Fill

전체 너비를 사용하며 각 세그먼트가 균등하게 채워집니다.

<ComponentPreview code={`<SegmentedControl size="md" fullWidth items={[
  { label: '전체', current: true },
  { label: '진행중' },
  { label: '완료' },
  { label: '취소' },
]} />`}>
  <div style={{ display: 'flex', flexDirection: 'column', gap: 16 }}>
    <SegmentedControl size="sm" fullWidth items={fourItems} />
    <SegmentedControl size="md" fullWidth items={fourItems} />
  </div>
</ComponentPreview>

---

## Number of segments

2개부터 8개까지 자유롭게 구성할 수 있습니다. 가독성을 위해 보통 5개 이하를 권장합니다.

<ComponentPreview code={`<SegmentedControl size="md" items={[
  { label: '전체', current: true },
  { label: '월간' },
  { label: '주간' },
  { label: '일간' },
  { label: '시간별' },
]} />`}>
  <div style={{ display: 'flex', flexDirection: 'column', gap: 16, alignItems: 'flex-start' }}>
    <SegmentedControl size="md" items={[
      { label: 'On', current: true },
      { label: 'Off' },
    ]} />
    <SegmentedControl size="md" items={threeItems} />
    <SegmentedControl size="md" items={fourItems} />
    <SegmentedControl size="md" items={fiveItems} />
    <SegmentedControl size="sm" items={eightItems} />
  </div>
</ComponentPreview>

## Sizes

`sm`과 `md` 두 가지 사이즈를 지원합니다.

<ComponentPreview code={`<SegmentedControl size="sm" items={items} />
<SegmentedControl size="md" items={items} />`}>
  <div style={{ display: 'flex', flexDirection: 'column', gap: 16, alignItems: 'flex-start' }}>
    <SegmentedControl size="sm" items={fourItems} />
    <SegmentedControl size="md" items={fourItems} />
  </div>
</ComponentPreview>
