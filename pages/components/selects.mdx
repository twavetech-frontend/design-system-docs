# Selects

Select는 사용자가 정해진 목록에서 값을 선택하는 입력 컴포넌트입니다. 라벨, 컨트롤(input), 힌트 텍스트로 구성되며, 펼친 상태에서는 아래에 메뉴 패널이 부착됩니다.

Figma: `Base components/Select` (3281:377673), `Base components/_Select menu item` (3281:380429)

---

## 사이즈

`md`, `sm` 두 사이즈를 제공합니다. 둘 다 동일한 무테두리 alpha-black-3 배경이며, 차이는 좌우 패딩과 chevron 아이콘 크기입니다.

- **md** — `padding: 6px 14px` · chevron 20px
- **sm** — `padding: 6px 12px` · chevron 16px

<ComponentPreview direction="column" code={`<Select size="md" label="Team member" required helpIcon value="John Lee" supportingText="@john" hint="This is a hint text to help user." />
<Select size="sm" label="Team member" required helpIcon value="John Lee" supportingText="@john" hint="This is a hint text to help user." />`}>

  <Select size="sm" label="Team member" required helpIcon value="John Lee" supportingText="@john" hint="This is a hint text to help user." />
</ComponentPreview>

## 상태

`Default`, `Placeholder`, `Focused`, `Disabled`, `Open` 5가지 상태가 있습니다. Focused/Open 상태는 2px 브랜드 보더와 alpha-black-5 배경을 사용합니다.

<ComponentPreview direction="column" code={`<Select label="Placeholder" required helpIcon placeholder="Select team member" hint="This is a hint text to help user." />
<Select label="Default" required helpIcon value="John Lee" supportingText="@john" hint="This is a hint text to help user." />
<Select label="Focused" required helpIcon state="focused" value="John Lee" supportingText="@john" hint="This is a hint text to help user." />
<Select label="Disabled" required helpIcon state="disabled" value="Select team member" hint="This is a hint text to help user." />`}>
  <Select label="Placeholder" required helpIcon placeholder="Select team member" hint="This is a hint text to help user." />
  <Select label="Default" required helpIcon value="John Lee" supportingText="@john" hint="This is a hint text to help user." />
  <Select label="Focused" required helpIcon state="focused" value="John Lee" supportingText="@john" hint="This is a hint text to help user." />
  <Select label="Disabled" required helpIcon state="disabled" value="Select team member" hint="This is a hint text to help user." />
</ComponentPreview>

### Open 상태

`state="open"`은 컨트롤 하단 모서리를 평평하게 만들어 메뉴 패널이 부착될 자리를 시각적으로 만듭니다. 실제 메뉴는 `<SelectMenu>`로 별도 렌더링합니다.

<ComponentPreview direction="column" code={`<Select label="Team member" required helpIcon state="open" value="John Lee" supportingText="@john" />
<SelectMenu>
  <SelectItem>Phoenix Baker</SelectItem>
  <SelectItem selected supportingText="@john">John Lee</SelectItem>
  <SelectItem>Lana Steiner</SelectItem>
  <SelectItem>Demi Wilkinson</SelectItem>
  <SelectItem>Candice Wu</SelectItem>
</SelectMenu>`}>
  <div style={{ display: 'flex', flexDirection: 'column' }}>
    <Select label="Team member" required helpIcon state="open" value="John Lee" supportingText="@john" />
    <SelectMenu>
      <SelectItem>Phoenix Baker</SelectItem>
      <SelectItem selected supportingText="@john">John Lee</SelectItem>
      <SelectItem>Lana Steiner</SelectItem>
      <SelectItem>Demi Wilkinson</SelectItem>
      <SelectItem>Candice Wu</SelectItem>
    </SelectMenu>
  </div>
</ComponentPreview>

### Search

`search` 프롭을 사용하면 검색 아이콘이 표시되고 chevron 대신 input 필드로 동작합니다. 우측에 `shortcut` 으로 키보드 단축키 뱃지를 붙일 수 있습니다.

<ComponentPreview direction="column" code={`<Select label="Search" required helpIcon search placeholder="Search" shortcut="⌘K" />`}>
  <Select label="Search" required helpIcon search placeholder="Search" shortcut="⌘K" />
</ComponentPreview>

### Tags

`tags` 배열을 전달하면 검색 아이콘과 함께 브랜드 컬러 칩이 렌더됩니다. 각 항목은 문자열이거나 `{ label, avatar }` 형태입니다.

<ComponentPreview direction="column" code={`<Select
  label="Search"
  required
  helpIcon
  shortcut="⌘K"
  tags={[
    { label: "John", avatar: "https://i.pravatar.cc/80?img=47" },
    { label: "Phoenix", avatar: "https://i.pravatar.cc/80?img=32" },
  ]}
  placeholder="Lana"
/>`}>
  <Select
    label="Search"
    required
    helpIcon
    shortcut="⌘K"
    tags={[
      { label: "John", avatar: "https://i.pravatar.cc/80?img=47" },
      { label: "Phoenix", avatar: "https://i.pravatar.cc/80?img=32" },
    ]}
    placeholder="Lana"
  />
</ComponentPreview>

## Select Menu Item

드롭다운 패널에 사용하는 메뉴 아이템입니다. 값(`label`)과 보조 텍스트(`supportingText`)를 한 줄에 표시하며, `selected`이면 우측에 체크 아이콘이 추가됩니다.

### 기본 / 선택 / 비활성

<ComponentPreview direction="column" code={`<SelectItem>Phoenix Baker</SelectItem>
<SelectItem supportingText="@john">John Lee</SelectItem>
<SelectItem selected supportingText="@john">John Lee</SelectItem>
<SelectItem disabled supportingText="@john">John Lee</SelectItem>`}>
  <SelectMenu>
    <SelectItem>Phoenix Baker</SelectItem>
    <SelectItem supportingText="@john">John Lee</SelectItem>
    <SelectItem selected supportingText="@john">John Lee</SelectItem>
    <SelectItem disabled supportingText="@john">John Lee</SelectItem>
  </SelectMenu>
</ComponentPreview>

### 아이콘 / 아바타 / Dot

<ComponentPreview direction="column" code={`<SelectItem icon={<MailIcon />} supportingText="@john">John Lee</SelectItem>
<SelectItem avatarSrc="https://i.pravatar.cc/80?img=47" supportingText="@john">John Lee</SelectItem>
<SelectItem dot="#17b26a">Active</SelectItem>
<SelectItem dot="#f04438" selected>Error</SelectItem>`}>
  <SelectMenu>
    <SelectItem icon={<MailIcon />} supportingText="@john">John Lee</SelectItem>
    <SelectItem icon={<SettingsIcon />} selected supportingText="@phoenix">Phoenix Baker</SelectItem>
    <SelectItem avatarSrc="https://i.pravatar.cc/80?img=47" supportingText="@john">John Lee</SelectItem>
    <SelectItem avatarSrc="https://i.pravatar.cc/80?img=32" selected supportingText="@phoenix">Phoenix Baker</SelectItem>
    <SelectItem dot="#17b26a">Active</SelectItem>
    <SelectItem dot="#f04438" selected>Error</SelectItem>
  </SelectMenu>
</ComponentPreview>
