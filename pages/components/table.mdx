# Table

테이블은 정렬·필터링·선택 가능한 행 형태로 데이터를 표시하는 컴포넌트입니다. 사용자, 회사, 파일, 거래 내역 등 구조화된 데이터를 한눈에 보여줄 때 사용합니다.

Figma 컴포넌트: `Table`, `Table header cell`, `Table cell`

---

## Default

가장 기본적인 형태로, `columns` 배열에 헤더 정보를 정의하고 `data` 배열로 행을 채웁니다.

<ComponentPreview code={`<Table columns={teamColumns} data={teamData} />`}>
  <Table size="md" columns={teamColumns} data={teamData} />
</ComponentPreview>

---

## Selectable rows

`selectable` 속성으로 행 선택 체크박스를 추가할 수 있습니다. 헤더 체크박스는 전체 선택 / 전체 해제로 동작합니다.

<ComponentPreview code={`<Table selectable columns={teamColumns} data={teamData} />`}>
  <Table size="md" selectable columns={teamColumns} data={teamData} />
</ComponentPreview>

---

## Sizes

`sm`, `md`, `lg` 세 가지 크기를 지원합니다.

<ComponentPreview code={`<Table size="sm" ... />
<Table size="md" ... />
<Table size="lg" ... />`}>
  <div style={{ display: 'flex', flexDirection: 'column', gap: 24 }}>
    <Table size="sm" columns={teamColumns} data={teamData.slice(0, 3)} />
    <Table size="md" columns={teamColumns} data={teamData.slice(0, 3)} />
    <Table size="lg" columns={teamColumns} data={teamData.slice(0, 3)} />
  </div>
</ComponentPreview>

---

## Custom cell render

각 컬럼에 `render` 함수를 두면 셀을 자유롭게 커스터마이즈할 수 있습니다. `UserCell`, `StatusBadge`는 헬퍼 컴포넌트로 함께 export 됩니다.

<ComponentPreview code={`const columns = [
  { key: 'name', header: '파일명' },
  { key: 'size', header: '크기', align: 'right' },
  { key: 'status', header: '상태', render: (row) => <StatusBadge variant={row.statusVariant}>{row.status}</StatusBadge> },
  { key: 'uploaded', header: '업로드일', align: 'right' },
]`}>
  <Table size="md" columns={fileColumns} data={fileData} />
</ComponentPreview>

---

## Sortable header

컬럼에 `sort` 속성으로 정렬 화살표를 표시할 수 있습니다 (`'asc'`, `'desc'`, `'none'`).

<ComponentPreview code={`const columns = [
  { key: 'name', header: '이름', sort: 'desc' },
  { key: 'team', header: '팀', sort: 'none' },
  { key: 'role', header: '역할' },
]`}>
  <Table
    size="md"
    columns={[
      { key: 'name', header: '이름', sort: 'desc', render: (r) => <UserCell src={r.avatar} name={r.name} email={r.email} /> },
      { key: 'team', header: '팀', sort: 'none' },
      { key: 'role', header: '역할' },
    ]}
    data={teamData.slice(0, 4)}
  />
</ComponentPreview>

---

## Empty state

데이터가 없을 때는 `emptyText`로 안내 문구를 표시합니다.

<ComponentPreview code={`<Table columns={teamColumns} data={[]} emptyText="등록된 멤버가 없습니다." />`}>
  <Table size="md" columns={teamColumns} data={[]} emptyText="등록된 멤버가 없습니다." />
</ComponentPreview>

---

## StatusBadge

테이블 셀 안에서 자주 사용되는 상태 뱃지입니다. `success`, `warning`, `error`, `gray`, `brand`, `info` 6가지 variant를 지원합니다.

<ComponentPreview code={`<StatusBadge variant="success">활성</StatusBadge>
<StatusBadge variant="warning">대기</StatusBadge>
<StatusBadge variant="error">오류</StatusBadge>`}>
  <div style={{ display: 'flex', gap: 8, flexWrap: 'wrap' }}>
    <StatusBadge variant="success">활성</StatusBadge>
    <StatusBadge variant="warning">대기</StatusBadge>
    <StatusBadge variant="error">비활성</StatusBadge>
    <StatusBadge variant="gray">취소</StatusBadge>
    <StatusBadge variant="brand">프리미엄</StatusBadge>
    <StatusBadge variant="info">정보</StatusBadge>
    <StatusBadge variant="success" dot={false}>점 없음</StatusBadge>
  </div>
</ComponentPreview>
